This project contains data that was collected over a 19 year period by researchers at Envionment and Climate Change Canada (ECCC) within and around the St. Denis National Wildlife Area (NWA). Birds in this dataset were banded and often nasal tagged in the NWA from 1982 - 2000. These marked birds were then resighted various times until 2001. This dataset was digitzed and cleaned through the Living Data Project data rescue hackathon and data rescue internship with the help of graduate intern Rodrigo Martin de Oliveira and undergraduate assistant Abigail Sadowski, and under the supervision of Dr. Jessica Reemeyer and Dr. Bruno Soares (University of Regina) and Mercy Harris (ECCC). Questions about this dataset can be sent to Jessica (jessica.reemeyer[at]mail.mcgill.ca).

This project contains five spreadsheets in .csv format. The banding_info.csv file contains species and banding information for individual birds. The species_codes.csv file provides full scientific and common species names for the 4 digit-codes used in the banding_info spreadsheet. The resightings.csv file contains records of resightings for the birds listed in banding_info. The pond_coords.csv file provides latitude and longitude coordinates for some of the resighting records. Please consult the data_dictionary.csv file for a full description of all of the variables in each spreadsheet.
